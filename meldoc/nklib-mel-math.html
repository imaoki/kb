    proc float[] getPlanarAimMatrix

    指定した2点（始点、終点）と中点行列の平面上で、始点-終点の線分と中点の最近接交点から中点方向へオフセットされたエイム行列を生成する。

    パラメータ
    $start
    <float[3]>

    始点。

    パラメータ
    $end
    <float[3]>

    終点。

    パラメータ
    $midMatrix
    <float[16]>

    中点のワールド行列（位置およびローカル軸の取得に使用）。

    パラメータ
    $aimAxis
    <int>

    最近接交点から中点方向への軸。

    パラメータ
    $upAxis
    <int>

    3点の平面上にあるアップベクトル。
    始点-終点-中点が直線状の場合は、$aimAxisに該当する中点のローカル軸方向にオフセットする。

    パラメータ
    $offsetMode
    <int>

    オフセット方式。

    値 方式 説明
    0 絶対値 最近接交点から$offsetで指定した距離だけオフセットする。
    1 比率 最近接交点から始点-終点の距離に$offsetを乗じた距離だけオフセットする。
    パラメータ
    $offsetBase
    <float>

    比率モードの基準長。
    -1の場合は始点-終点の距離を基準にする。
    正の値を指定した場合はその値を基準にする（例: 上腕＋前腕の全長）。
    絶対値モードでは無視される。

    パラメータ
    $offset
    <float>

    オフセット量。$offsetModeにより意味が変わる。

    戻り値
    <float[16]>
    補足

    取得した行列から位置、回転、スケールを抽出するには以下のようにする。

    float $armLength = 30.0;
float $start[] = `xform -q -t -ws $startNode`;
float $end[] = `xform -q -t -ws $endNode`;
float $tm[] = `xform -q -ws -m $midNode`;

// 絶対値モード
float $m[] = getPlanarAimMatrix($start, $end, $tm, 1, 0, 0, -1.0, 5.0);

// 全長基準の比率モード
float $m2[] = getPlanarAimMatrix($start, $end, $tm, 1, 0, 1, $armLength, 0.5);

// 始点-終点距離を基準にした比率モード
float $m3[] = getPlanarAimMatrix($start, $end, $tm, 1, 0, 1, -1.0, 0.5);
