    proc string[] collectDescendantNodes

    パラメータ
    $n
    <string>
    パラメータ
    $parentNode
    <string>

    親ノード。

    パラメータ
    $types
    <string[]>

    ノードタイプの配列。空配列を指定した場合は全て取得する。

    パラメータ
    $isExact
    <boolean>

    派生型を含めない場合はtrue、派生型を含める場合はfalse。

    パラメータ
    $script
    <string>

    ノードへの処理を定義したグローバルプロシージャの名前。
    このプロシージャの戻り値は$typesの影響を受ける。
    空文字列の場合は何もしない。

    構文
    global proc string[] appendSuffix(string $sourceId, string $parentId) {
  string $sourceNodes[] = `ls $sourceId`;
  rename $sourceNodes[0] ($sourceNodes[0] + "_suffix");
  return {$sourceId, $sourceId};
}
    引数
    引数 型 説明
    $sourceId string 現在の処理対象ノードのUUID
    $parentId string 前回処理済みの親ノードのUUID
    戻り値
    配列要素 説明
    string[0] 処理対象ノードに置き換わるノードのUUID
    string[1] 次に子ノードを探索するノードのUUID

    ノードのリネーム等、対象が変わらない場合は戻り値の1番目と2番目の要素は同じにする。
    ノードの複製等で対象が変わる場合は1番目に複製したノードのUUID、2番目に元のノードのUUIDを指定する。

    戻り値
    <string[]>
